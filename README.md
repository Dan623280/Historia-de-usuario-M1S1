# Product Registration Program

## Description
This project is a simple Python program for registering products.  
It is designed to ask the user for product information and calculate the total cost.

The program is simple and easy to use.  
It is useful for learning how to work with variables, user input, and error handling in Python.

## How it works

1. The program displays a welcome message to the user.
2. The system asks for the product name.
3. The program asks for the unit price of the product.
4. The system asks for the quantity of the product.
5. The program calculates the total cost using the price and quantity.
6. The system prints the product information and the total cost.
7. If the user enters an invalid value, the program shows an error message.

> The program is currently running in a continuous loop and is waiting for user input.
